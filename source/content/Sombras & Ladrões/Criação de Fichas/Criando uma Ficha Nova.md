## Definindo Atributos
Para definir os valores de **Auto Controle**, **Constituição**,  **Destreza**, **Educação**,  **Empatia**,  **Força**, **Inteligência** e **Tamanho**, role 4d6 e some os 3 maiores resultados, os valores podem então ser distribuídos da forma que preferir entre estes atributos.

Para definir o valor de **Movimento**, role 1d4 e some a 4. **Saltar** e **Escalar** será sempre igual a metade do seu valor de **Movimento** e **Pular** será igual a um quarto.

Depois pontos iguais ao valor de **Poder** do personagem podem ser distribuídos para qualquer atributo.

## Status
### Iniciativa
O valor base de **Iniciativa** é igual a soma dos pontos de **Destreza** e **Inteligência**
	Formula: **Iniciativa** = **Destreza** + **Inteligência**
### Vida Máxima
O valor base de **Vida Máxima** é igual ao dobro dos pontos de **Constituição** somado aos pontos de **Tamanho**
	Formula: **Vida Máxima** = (**Constituição**  * 2 ) + **Tamanho**

### Bônus de Dano
Para achar o bonus de dano, some **Força** e **Tamanho**, seu **Bônus de Dano** irá seguir a tabela:

| **Força** + **Tamanho** | **Bônus de Dano** |
| :---------------------: | :---------------: |
|           1-5           |         1         |
|          6-10           |        1d2        |
|          11-15          |        1d3        |
|          16-20          |        1d4        |
|          21-25          |        1d6        |
|          26-30          |        1d8        |
|          31-35          |       1d10        |
|          36-39          |       1d12        |
|           40            |        2d6        |
